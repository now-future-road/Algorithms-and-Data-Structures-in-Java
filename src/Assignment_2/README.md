Project Structure

The core logic is located in the src/Assignment_2 directory:

MyList.java: A custom interface defining the standard operations for list-based structures.

MyArrayList.java: A dynamic array implementation that automatically resizes.

MyLinkedList.java: A doubly linked list implementation for efficient insertions and deletions.

MyStack.java: A Last-In-First-Out (LIFO) structure built using custom list logic.

MyQueue.java: A First-In-First-Out (FIFO) structure.

MyMinHeap.java: A binary min-heap implementation for priority-based operations.

Main.java: Entry point for testing and demonstrating the functionality of each structure.

Features Implemented

Generic types to support any data type (<T>).

Heapify and bubble-up/down logic in MyMinHeap.

Standard operations: add(), get(), remove(), size(), and isEmpty().
